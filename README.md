# Checkers Website
This project was created as part of a CSCI 130 Web Development course at Fresno State. 
To start, create an account! From there, you can browse the site itself.
You can play on either 8x8 or 10x10 boards, with a bot or with another player.
_Feel free to mess around with the colors of the board and checkers!_
The leaderboard should show top games from members on the site.

# Check it out!
To use the site, check out:
https://checkers-ccdw.onrender.com

# Created By:
This project was done by Amanda Balker over the span of about 4 weeks. I started off with making the entirety of the site itself, focusing on design, redirection, and making sure everything fits!
I then started by making the game for the 8 by 8 board. Using this script, I copied the same exact template to make the 10 by 10 board!
After setting up the website itself and the games, I implemented the php files, making sure the data would save to a database and show up in the leaderboard as well!
This was originally done with XAMPP, then transitioned into remote hosting. Some changes have been made since originally written years ago.
